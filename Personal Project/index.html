About my personal project

On this webpage I will be talking about what my personal project was, why I chose it, what I learned

What my personal project was.

For my personal project I decided to learn something completely new. I decided that I would learn how to code a website, and even with absolutely no knowledge of this before hand I was still able to accomplish my task. Choosing this topic taught me many things and made me realize how much time and effort really goes into the creation of any webpage or anything else that is coded.

Why I decided I would learn to code

A short summary of why I learned to code is:

  1. I thought it was interesting.
  2. It was something that I knew nothing about.
  3. I wanted to learn something new that I didn't know anything about.

I mainly decided to code because, recently I have started to learn more about computers and how they work and how they are designed, and usually when I think about computers I think about coding and programming. While I was learning all this information a lot of sources mentioned coding. This caused me to start researching: what coding is, how it works and most importantly how to code.

I think coding is interesting because it is essentially another language that is used to make almost everything on the internet. To me it is just fascinating how just a few letters and numbers can make a website or even a video game.
